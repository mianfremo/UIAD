Verificar Canal

Este boton se encarga realizar las mismas verificaciones que el diagnostico, sin embargo, el boton de verificar canal solo comprueba el canal seleccionado, de tal forma que se puede estudiar con mas detalle si existe una falla en alguno de los canales.


Una vez presionado el boton, se despliega una ventana de confirmacion con los datos del canal seleccionado, la cual solo es mostrada si el mismo fue configurado previamente y la conexion con UIAD es exitosa. Si se trata de un canal de entrada, la verificacion devuelve la lectura actual del canal con el respectivo escalamiento configurado, mientras que si el canal seleccionado es una salida, dicha salida es activada y muestra un mensaje de confirmacion en la pantalla para verificar si efectivamente se activo, caso afirmativo la UIAD procede a apagar dicha salida.
