Diagnostico

Este boton se encarga de realizar las verificaciones necesarias a cada uno de los canales que se encuentren habilitados, al momento de presionar el boton, una ventana de confirmacion es desplegada con toda la informacion previamente configurada de cada uno de los canales para posteriormente comenzar el diagnostiico de los mismos.


Es importante destacar que solo puede ser realizado el diagnostico si la conexion con la UIAD es exitosa y si los canales estan previamente configurados, ya que si no se establecen algunos parametros no es posible realizar las verificaciones necesarias.
