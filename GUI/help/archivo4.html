Exportar Archivo

Con el uso de este boton es posible exportar un archivo de configuracion a cualquier ubicacion que se desee, cuando el mismo es presionado una ventana de seleccion de archivos es abierta y permite guardar el archivo actual de configuracion en una ubicacion determinada.



Una vez seleccionada la ubicacion, escriba un nombre para el archivo a exportar y presione el boton "save", automaticamente el archivo se guardara con la extension adecuada y podra ser empleado en cualquier otra implementacion de la UIAD.
