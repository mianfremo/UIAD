Cargar Archivo

Con el uso de este boton es posible cargar un archivo de configuracion desde cualquier ubicacion que se desee, cuando el mismo es presionado una ventana de seleccion de archivos es abierta y permite escoger el archivo de configuracion a importar.



Una vez seleccionado el archivo, presione el boton "open" y se cargara automaticamente la informacion contenida en ese archivo. Si el archivo no contiene la extension correcta, se mostrara un mensaje de error y se conservara el archivo actual de configuracion.
